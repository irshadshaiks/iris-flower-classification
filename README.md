# Iris Flower Classification

Internship Task 1 - Machine Learning Classification using the Iris Dataset

## Models
- K-Nearest Neighbors
- Logistic Regression  
- Decision Tree

## Results
All models achieved 93.33% test accuracy and ~96.67% cross-validated accuracy.

pip install -r requirements.txt
